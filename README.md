# KEERO Landing Page

Landing page for KEERO - Local AI Companion for Creators.

## Development

```bash
npm install
npm run dev
```

The landing page will be available at `http://localhost:3000`

## Build

```bash
npm run build
```

The built files will be in the `dist` folder.

## Preview Production Build

```bash
npm run preview
```

## Features

- Modern, clean design matching the KEERO app style
- Responsive layout
- Uses the same component system and CSS variables as the main app
- Hero section with call-to-action
- Features showcase
- Footer with links

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Lucide React (icons)
